Handoff Notes for Meridian IT
No persistence layer
Data is loaded from static JSON files at server startup. Tasks and purchase orders created at runtime are lost on restart. Production deployment requires a real database (PostgreSQL recommended).
No authentication
The current build has no auth — useAuth() returns a hardcoded mock user. Meridian SSO integration is required before production.
Dev-only CORS
CORS is open (allow_origins=["*"]). Restrict to the production frontend domain before deployment.
