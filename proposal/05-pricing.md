# Pricing

**RFP #MC-2026-0417 — Meridian Components**

---

## Summary

We are proposing a **fixed-fee engagement** in two tiers: Required scope (R1–R4) and an optional add-on for Desired items (D1–D3). Fixed fee means no billing surprises — what you see below is what you pay, regardless of how many hours it takes us.

## Payment Schedule

We bill against milestones, not calendar months. For the Required scope:

| Milestone | Amount |
|---|---|
| Contract execution | $16,200 (30%) |
| Phase 1 complete — R4 delivered and accepted | $10,800 (20%) |
| Phase 2 complete — R1 + R3 delivered and accepted | $16,200 (30%) |
| Phase 3 complete — R2 delivered and accepted | $10,800 (20%) |
| **Total** | **$54,000** |

For Desired items (if added): same 30/20/30/20 structure, billed as each item is delivered and accepted.

---

## What Is Included

- All development work
- Code committed to Meridian's repository with clear commit history
- Playwright test suite with documentation
- Architecture documentation (`architecture.html`)
- Weekly status touchpoints with Operations and IT
- 30-day warranty period after final delivery: we fix any defect in delivered work at no charge

## What Is Not Included

- Infrastructure or hosting changes
- Database migration (current JSON-file data layer is retained)
- External system integrations (ERP, procurement, etc.)
- Changes to requirements after phase sign-off (handled via change order)

---

## Assumptions

1. Meridian provides the eight-defect list for R1 at kickoff
2. Meridian provides brand guide for D1 by start of Week 7
3. Meridian stakeholders available for 1-hour weekly reviews
4. No new regulatory or compliance requirements emerge during the engagement
5. Work is performed on Meridian's existing codebase as provided; no environment rebuild required
